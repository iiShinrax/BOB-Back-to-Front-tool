# BOB AI Full-Stack Generator

أداة لأتمتة بناء الباك إند وربطه بالفرونت إند باستخدام الذكاء الاصطناعي.

## طريقة التشغيل للتيم:
1. استنسخ المشروع (Clone).
2. سو ملف `.env` وحط فيه مفتاحك: `OPENROUTER_API_KEY=your_key_here`
3. شغل الأداة: `demo.js`
(الأداة بتحمل المكتبات من نفسها وتفتح لك نافذة تختار منها ملف الفيجما).
